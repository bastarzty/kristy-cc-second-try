Experimental Clock

In this project we design and code an experimental clock that expresses a novel way of tracking time.
I came out with about three initial ideas. So the first idea was about heart bumping along with human breathing rhythm. And the second one was about student fall into sleep at class,
I wanted to create the effect of an eye close gradually and once the mouse click/pressed you can wakes him up. And the last idea was this one about watering tree.
I initially wanted to try the eye but i have no idea of how to achieve my concept so that I decided to change to the tree one. I was using the 'countdown = interval'to set up the countdown part so to make the apple falls when you are not clicking
